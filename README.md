# Чат-бот с системой определения намерений
Данный проект представляет собой эволюцию учебного чат-бота. Бот умеет отвечать на вопросы о времени, дате, годе, погоде, поддерживает приветствия и прощания.
## _Этапы разработки:_
### 1. Первая версия бота (21 февраля)
   - определение намерений с помощью регулярных выражений,
   - добавление обработчиков (приветствие, прощание, время, дата, год, "как дела", калькулятор),
   - логирование диалогов в текстовый файл.
### 2. Добавление погоды и базы данных (28 февраля)
   - интеграция с Weatherstack API для получения сведений о погоде
   - запрос погоды по шаблону "погода в {город}"
   - логирование перенесено из файла в SQLite базу данных
   - сохранение пользователей и истории сообщений
### 3. Использование spaCy для обработки естественного языка (7 марта)
   - улучшение обработки естественного языка
   - интеграция библиотеки spaCy с моделью ru_core_news_sm
   - реализован класс NLPProcessor для работы с текстом
   - добавлено извлечение городов через NER (Named Entity Recognition)
   - запросы погоды обрабатываются через NLP: бот сам находит город в тексте
### 4. Dialog Manager и FSM (14 марта)
   - реализован конечный автомат с состояниями START и WAIT_CITY
   - создан Dialog Manager для хранения состояния каждого пользователя
   - при запросе "какая погода" бот спросит "в каком городе"
### 5. ML подход (21 марта)
   - создан датасет с 220+ примерами для 7 интентов: greeting, goodbye, time, date, year, how_are_you, weather
   - реализована предобработка текста (лемматизация, удаление стоп-слов)
   - выполнена TF-IDF векторизация
   - обучена модель
   - добавлена проверка уверенности (confidence)
### 6. Word Embeddings (28 марта)
   - замена модели ru_core_news_sm на ru_core_news_md
   - получение вектора текста через doc.vector
   - переобучение модели на новых векторах
### 7. Переход к Contextual Embeddings: Fine-Tuning BERT (4 апреля)
   - использована предобученная модель DeepPavlov/rubert-base-cased
   - бот переведён на BERT – замена spaCy embeddings
